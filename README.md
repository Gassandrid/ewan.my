# My Digital Garden

This is probably the project that I use the most / put the most effort into.It is a digital garden that I use to keep track of my notes, ideas, and projects. It is built using [Quartz](https://quartz.jzhao.xyz/) and [Obsidian](https://obsidian.md/)...

While I am always going to be a "computer scientist" at heart, my focus is now on Computational Neuroscience and the Mathematical modeling that comes with it. Because of that, this project is focused on that area, but I also have notes on other topics that I find interesting.
